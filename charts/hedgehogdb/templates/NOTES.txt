HedgehogDB has been installed.

1. Ensure the HedgehogDB image is available (e.g. load into cluster):
   docker build -t hedgehogdb:latest .
   docker save hedgehogdb:latest | sudo k3s ctr images import -

2. Cluster forms automatically: pods start in order (0, 1, 2); each joins via seed nodes (headless DNS).

3. Access the API and Web UI:
   - In-cluster: http://{{ include "hedgehogdb.fullname" . }}-api.{{ .Release.Namespace }}.svc.cluster.local:8080
   - Ingress: add to /etc/hosts: {{ .Values.ingress.host }} → <ingress-ip>
   - Or port-forward: kubectl port-forward svc/{{ include "hedgehogdb.fullname" . }}-api 8080:8080 -n {{ .Release.Namespace }}

4. Check cluster status: curl http://localhost:8080/api/v1/cluster/status
